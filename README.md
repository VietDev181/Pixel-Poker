# 🃏 Pixel Poker

**Pixel Poker** là một tựa game solitaire mang phong cách pixel cổ điển, nơi người chơi sắp xếp các lá bài theo thứ tự hợp lệ để dọn sạch bàn chơi. Game mang lại trải nghiệm thư giãn nhưng vẫn đủ thử thách với lối chơi quen thuộc và dễ tiếp cận.

---

## 🎮 Gameplay

- Sắp xếp các lá bài theo quy tắc solitaire
- Dọn sạch toàn bộ bàn chơi để chiến thắng
- Tính toán chiến lược để tối ưu nước đi
- Trải nghiệm nhẹ nhàng, phù hợp chơi mọi lúc
